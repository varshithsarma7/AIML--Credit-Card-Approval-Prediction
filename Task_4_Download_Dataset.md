# Download the Dataset

## Dataset Used

The Credit Card Approval Prediction System uses the **Credit Card Approval Prediction Dataset** obtained from Kaggle.

### Dataset Files

- application_record.csv
- credit_record.csv

### Dataset Description

**application_record.csv**
Contains applicant information such as:
- Gender
- Income Type
- Annual Income
- Education
- Family Status
- Occupation
- Housing Type
- Employment Details

**credit_record.csv**
Contains applicants' credit history, including:
- ID
- Monthly Record
- Credit Status

The two datasets are linked using the common **ID** column.

### Dataset Source

Kaggle:
https://www.kaggle.com/datasets/rikdifos/credit-card-approval-prediction

### Dataset Format

- File Type: CSV
- Number of Files: 2
- Encoding: UTF-8

### Purpose

The dataset is used to:
- Analyze applicant information
- Study credit history
- Build machine learning models
- Predict credit card approval status
