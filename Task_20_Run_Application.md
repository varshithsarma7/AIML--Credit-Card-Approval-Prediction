# Task 20: Run the Application

## Running the Credit Card Approval Prediction System

After completing data preprocessing, feature engineering, model training, and developing the Flask web application, the application is executed locally.

### Steps to Run

1. Open the project folder.
2. Open a terminal or command prompt.
3. Install the required libraries.

```bash
pip install -r requirements.txt
```

4. Run the Flask application.

```bash
python app.py
```

5. Open a web browser.

6. Visit:

```
http://127.0.0.1:5000/
```

7. Enter the applicant details in the form.

8. Click the **Predict** button.

9. The application displays the prediction as:

- Credit Card Approved
- Credit Card Rejected

## Expected Output

The Flask server starts successfully and hosts the Credit Card Approval Prediction web application on the local machine. Users can enter applicant details through the web interface and receive the prediction generated by the trained machine learning model.
